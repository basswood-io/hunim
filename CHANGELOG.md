## Features
 - Feed pages now automatically inject a `<link rel="alternate" type="application/rss+xml">` tag for RSS autodiscovery
 - Feed index pages can use a `{{ .PostList }}` placeholder that is replaced with the autogenerated, newest-first list of posts
 - RSS feeds now include item descriptions

## Fixes
 - Fixed crashes on empty and malformed input
 - HTML meta-tag values are now escaped in generated pages
 - RSS output is now properly escaped
 - Frontmatter is now parsed only once instead of repeatedly
